##Versão de Testes##

Tecnologias Utilizadas
    Node.Js v20.10.0 ou superior (LTS)
    React: v19.1.1
    Vite: v7.1.0

Instalação:
    Navegue até a pasta do projeto:
        cd dashlab

    Instale as dependências:
        npm install

Como rodar:
    Após a instalação, você pode iniciar o servidor de desenvolvimento com o seguinte comando:
        npm run dev
